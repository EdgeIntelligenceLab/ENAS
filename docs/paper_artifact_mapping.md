# Paper Artifact Mapping

This document maps every figure and table in the paper to the script that produces it. After running experiments and parsing logs, all artifacts can be regenerated by running the listed script.

## Workflow

```
[1] Run experiments       → results/raw_logs/
[2] Parse logs to CSV     → results/parsed_csv/
[3] Generate tables       → results/summary_tables/
[4] Generate figures      → figures/output/
```

Run all three steps with:
```bash
python scripts/run_all_experiments.py --dataset vww
python scripts/run_all_experiments.py --dataset melanoma
python scripts/parse_logs.py
python scripts/generate_summary_tables.py
python figures/scripts/generate_all_figures.py
```

## Ablation Variants (Table 7)

To reproduce the ablation rows, run with the corresponding config from `configs/experiments/ablation.yaml`:

| Ablation Row | Config Variant | What Changes |
|---|---|---|
| ENAS-strategy (2D space) | `ablate_no_cell_space` | Uses 2D (k, c) search instead of cell-based |
| Ep=1, wacc=0.65 (baseline) | `ablate_proxy_epochs_1` + `ablate_eff_weight_high` | Reverts to ENAS v2 settings |
| + Ep=3 | `ablate_proxy_epochs_2/3` | Increases proxy epochs |
| + wacc=0.80 | (default) | Increases accuracy weight |
| + two-phase Stage 2 | (default) | Enables accuracy-first ranking |
| + warm seed | (default with seed_k, seed_c set) | Enables NanoNAS seeding |

Run with:
```bash
python scripts/run_ablation.py --ablation ablate_proxy_epochs_1
```
