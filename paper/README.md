# Paper

The paper PDF is omitted from this repository for double-blind review.
The full paper text and supplementary tables are submitted separately
through the SuRE @ IJCAI 2026 system.

The information needed for review is fully contained in the repository
README and in the figures generated by the figure scripts:

| Reference | How to obtain |
|---|---|
| Figure 1 (pipeline) | See `figures/scripts/plot_pipeline.py` |
| Figure 2 (Pareto)   | `python figures/scripts/plot_pareto.py` |
| Figure 3 (4-panel)  | `python figures/scripts/plot_heatmaps.py` |
| Tables 1–7          | `python scripts/generate_summary_tables.py` |

After de-anonymisation, the published version of the paper will be
included in this directory.
